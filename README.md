# Stocky

Stocky is a simple stock management app, with real-time stock tracking and inventory management.

## Get started

If you just cloned this codebase and didn't use `npm create convex`, run:

```
npm install
npm run dev
```

## Toolkit

Stocky is built with:
- Convex
- Next.js
- Convex Auth
- React
- Tailwind
